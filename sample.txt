HTML - HyperText Markup Language
- Used to send information over the internet
- Made up of tags

Header tags
<h1></h1>
<h2></h2>
<h3></h3>
<h4></h4>
<h5></h5>
<h6></h6>

List tags
-Ordered list - numbered
<ol></ol>

-Unordered list - bullet points
<ul></ul>

-List item
<li> Item number 1 </li>
<li> Item 2 </li>

-paragraphs
<p>Write here the content of
your paragraph</p>

-image
<img src="image-URL OR local location" 
  alt="alternate text">

- hyperlinks/anchor tags
a link to click on to navigate to another site/area on the page
<a href="website link">Hyperlink text</a>

CSS - Cascading Style Sheet
syntax structure-

 elementName { 	
 	propertyName: value; 
 }

 example- 

 h1 {
 	font-size: 14
 }
